# DOM-Pig-Game
Javascript Dom Game

## Rules

1. You can hold your current roll with the btn-hold
2. If a player rolls a 1 is turn of the next player
3. If a player rolls two 6 the score is reset
4. The player who reaches the score to win first win!!
